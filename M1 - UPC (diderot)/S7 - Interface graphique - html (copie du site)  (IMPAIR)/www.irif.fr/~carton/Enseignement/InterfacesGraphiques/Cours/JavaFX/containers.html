Conteneurs

La classe Region possède les propriétés suivantes qui dirigent l'organisation des composants.

  • width, height
  • minWidth, minHeight
  • maxWidth, maxHeight
  • prefWidth, prefHeight

La classe Pos contient les constantes suivantes :

  • TOP_LEFT, TOP_RIGHT, TOP_CENTER
  • CENTER_LEFT, CENTER_RIGHT, CENTER
  • BASELINE_LEFT, BASELINE_RIGHT, BASELINE_CENTER
  • BOTTOM_LEFT, BOTTOM_RIGHT, BOTTOM_CENTER

La classe HPos contient les constantes suivantes :

  • LEFT, CENTER, RIGHT

La classe VPos contient les constantes suivantes :

  • TOP, CENTER, BASELINE, BOTTOM

La classe Priority contient les constantes suivantes :

  • ALWAYS, SOMETIMES, NEVER

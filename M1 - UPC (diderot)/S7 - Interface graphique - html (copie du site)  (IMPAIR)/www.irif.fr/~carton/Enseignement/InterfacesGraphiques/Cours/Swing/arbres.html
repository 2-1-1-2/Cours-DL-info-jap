Arbres

Les arbres permettent de visualiser des données organisées de manière hiérarchique comme les fichiers et les répertoires ou comme les classes d'un langage orienté objet (comme Java par exemple). Ils sont manipulés en Swing avec la classe JTree. Les arbres offent les fonctionnalités suivantes.

  • Les arbres Swing utilisent l'architecture Modèle/Vue/Contrôleur. La gestion proprement dite est confiée à un objet qui implémente l'interface TreeModel. L'affichage proprement dit des objets est confié à un objet qui implémente l'interface TreeCellRenderer.
  • La gestion des éléments selectionnés est déléguée à un objet qui implémente l'interface TreeSelectionModel.
  • Les arbres implémentent l'interface Scrollable et peuvent donc être placées dans un conteneur JScrollPane avec des ascenseurs.

Les nœuds de l'arbre implémentent l'interface TreeNode.

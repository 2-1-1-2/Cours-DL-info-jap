Principaux clients X

  • Gestionnaires de fenêtres les plus classiques
    • twm fourni en standard avec X
    • mwm Motif (exemple de fichier de configuration .mwmrc)
    • fvwm (exemple de fichier de configuration .fvwmrc)
    • olwm OpenLook
    • tvtwm
  • Clients principaux
    • xdm : gestionnaire de session
    • xhost : gestionnaire d'accès au serveur
    • xauth : gestionnaire de clés de sécurité
    • xrdb : gestionnaire de la base de données des ressources
    • xkill : suppression d'un autre client
    • xterm : terminal avec un shell
    • resize : utilisation en eval `resize` dans un shell
  • Accessoires de bureau
    • xbiff : avertisseur de courrier
    • xclock : horloge
    • xmh : client pour le courrier
    • xload : affichage de la charge de la machine
    • xcalc : calculatrice
    • xman : consultation des manuels UNIX
    • xedit : éditeur de texte minimaliste
  • Préférences du clavier et de l'affichage
    • xset : contrôle le volume et la durée du bip, les paramètres de la souris et les font paths
    • xsetroot : contrôle le fond d'écran de la fenêtre racine et le curseur
    • xmodmap : contrôle des associations entre les touches physiques et les touches logiques
  • Utilitaires de fontes
    • xfd : affichage des caractères dans une fonte
    • xfontsel : outil de sélection de fontes
    • xlsfonts : affichage des fontes disponibles
  • Utilitaires graphiques
    • atobm : transformation d'ASCII en Bitmap
    • bmtoa : transformation de Bitmap en ASCII
    • bitmap : éditeur de bitmap
    • xmag : agrandisseur d'écran
    • xcol : affichage des noms des couleurs et des couleurs disponibles (fichier rgb.txt)
  • Utilitaires d'impression
    • xwd : dump de l'image d'une fenêtre
    • xwud : affichage de l'image dumpée d'une fenêtre
    • xpr : transformation de l'image produite par awd en PostScript ou un autre format
    • xdpr : xwd|xpr|lpr
  • Utilitaires d'information
    • xdpyinfo : information sur le serveur
    • xwininfo : information sur une fenêtre
    • xprop : propriétés d'une fenêtre
    • xlsclients : liste des clients X connectés au serveur
    • xlswins : liste des fenêtres actives
    • xlsatoms : liste des atomes du serveur
    • lstres : liste des ressources d'une widget
    • appres : liste des ressources d'une application
    • editres : éditeur des ressources d'une application
